Nori

En esta unidad veremos el caso nori.

El caso nori se utiliza para decir a quien o a que, es decir actúa como objeto indirecto.

Para declinar el objeto sobre el que recae la acción en el caso nori, primero tenemos que distinguir si es un nombre propio o un nombre comúm:

Para nombres propios utilizaremos la siguiente tabla:

Nori Kasua
Consonante -I

Manuel => ManuelI

Mikel => MikelI

Aitor => AitorrI

Vocal -RI

Amaia => AmaiaRI

Alba => AlbaRI

Salomé => SalomeRI

Y en el caso de los nombres comunes tendremos que distinguir si el nombre es singular o plural:

Nori Kasua
Singular -ARI

Txakur => TxakurrARI

Katu => KatuARI

Zuhaitz => ZuhaitzARI

Plural -EI

Txakur => TxakurREI

Katu => KatuEI

Zuhaitz => ZuhaitzEI

En este caso también tendremos que conjugar el auxiliar que acompaña al verbo dado que hay dos objetos que intervienen en la acción: sujeto(NORI) y objeto indirecto(NOR)

Para conjugarlo deberemos tener en cuenta si la el objeto que acompaña al verbo (el complemento indirecto) es singular o plural (esto lo veremos más claros con algunos ejemplos)

La tabla para conjugar el caso NOR-NORI es la siguiente

NOR-NORI
NORI Singular Plural
Niri zait zaizkit
Zuri zaizu zaizkizu
Hari zaio zaizkio
Guri zaigu zaizkigu
Zuei zaizue zaizkizue
Haiei zaie zaizkie

Para resumir la tabla quedaría así:

NOR-NORI
Niri zai(zki) -t
Zuri -zu
Hari -o
Guri -gu
Zuei -zue
Haiei -e
