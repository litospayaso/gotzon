Como puedes observar hay declinaciones (ari/zale) que se puede utlizar para construir profesiones a partir del verbo:

Aditza (verbo) Gaztelania Lanbidea (profesión)
Ikasi Estudiar Ikaslea (estudiante)
Saldu Vender Saltzailea (vendedor/a)
Programatu Programar Programatzailea (programador/a)
Gidatu Conducir Gidari (conductor/a)
Abesti Canción Abeslari (cantante)
Esto no funciona con todas las profesiones por ejemplo
actuar => antzeztu/lan egin
Actor => Aktorea
