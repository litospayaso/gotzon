Nora

En esta unidad veremos el caso nora.

El caso nora se utiliza para decir a dónde

Para nombres propios utilizaremos la siguiente tabla:

Nora Kasua
Consonante -ERA

Paris => ParisERA

Madril => MadrilERA

Eibar => EibaRRERA

Vocal -RA

Bermeo => BermeoRA

Bilbo => BilboRA

Donostia => DonostiaRA

Y en el caso de los nombres comunes tendremos que distinguir si el nombre es singular o plural:

Nora
Singular -(E)RA

Jatetxe (restaurante) => JatetxeRA

Ikastola => IkastolaRA

Zuhaitz => ZuhaitzERA

Plural -ETARA

Jatetxe (restaurante) => JatetxeETARA

Ikastola => IkastolETARA

Zuhaitz => ZuhaitzETARA

Recuerda que a la hora de poner poner los sufijos con las vocales a y e sigue la siguiente regla:
a + a => a
a + e => e
e + e => ee

Por último veamos los demostrativos y adverbios para el caso nora:

Demostrativos
Hau Honetara A este, esta, esto
Hori Horretara A ese, esa, eso
Hura Hartara A aquel, aquella, aquello
Hauek Hauetara A estas, estos
Horiek Horietara A esas, esos
Haiek Haietara A aquellas, aquellos
Adverbios
Hemen Hona Aquí
Hor Horra Ahí
Han Hara Allí
