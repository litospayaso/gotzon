Nondik

En esta unidad veremos el caso nondik.

El caso nora se utiliza para decir de dónde

Para nombres propios utilizaremos la siguiente tabla:

Nondik Kasua
Consonante -ETIK

Paris => ParisETIK

Madril => MadrilETIK

Eibar => EibaRRETIK

Vocal -TIK

Bermeo => BermeoTIK

Bilbo => BilboTIK

Donostia => DonostiaTIK

Y en el caso de los nombres comunes tendremos que distinguir si el nombre es singular o plural:

Nondik
Singular -(E)TIK

Jatetxe (restaurante) => JatetxeTIK

Ikastola => IkastolaTIK

Zuhaitz => ZuhaitzETIK

Plural -ETATIK

Jatetxe (restaurante) => JatetxeETATIK

Ikastola => IkastolETATIK

Zuhaitz => ZuhaitzETATIK

Recuerda que a la hora de poner poner los sufijos con las vocales a y e sigue la siguiente regla:
a + a => a
a + e => e
e + e => ee

Por último veamos los demostrativos y adverbios para el caso nora:

Demostrativos
Hau Honetatik De este, esta, esto
Hori Horretatik De ese, esa, eso
Hura Hartatik De aquel, aquella, aquello
Hauek Hauetatik De estas, estos
Horiek Horietatik De esas, esos
Haiek Haietatik De aquellas, aquellos
Adverbios
Hemen Hemendik De aquí
Hor Hortik De ahí
Han Handik De allí
