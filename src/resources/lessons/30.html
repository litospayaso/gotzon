Gerundio

Todos los verbos que hemos visto hasta ahora los hemos visto en su forma de "infinitivo" por ejemplo Hartu (coger), pero hay otra forma de tener ese verbo, que es el gerundio (Hartzen).

En esta unidad aprenderemos como hacer el gerundio de los verbos:

Terminación Transformación Ejemplo
-TU, -DU, -RI, -I Quitar la terminación y añadir -TZEN

Hartu (coger) => HarTZEN

Kendu (quitar) => KenTZEN

Etorri (venir) => EtorTZEN

Ibili (Andar) => IbilTZEN

-N Quitar la terminación y añadir -TEN

Egin (hacer) => EgiTEN

Egon (estar) => EgoTEN

-SI , -ZI Quitar la -I final y añadir -TEN (La Z y la S se quedan)

Erosi (Comprar) => ErosTEN

Hazi (Crecer) => HazTEN

-TZI, -TSI, -TXI, -ZTU, -STU Dejar la Z/S/X y añadir -TEN

Idatzi (escribir) => IdazTEN

Jaitsi (bajar) => JaisTEN

Itxi (cerrar) => IxTEN

Piztu (encender) => PizTEN

Abestu (Salir / Sacar) => AbesTEN

-KI Añadir -TZEN al final

Ebaki (Cortar) => EbakiTZEN

Eduki (Tener) => EdukiTZEN

Esto es el caso general, pero como siempre pueden haber excepciones, el mejor modo de aprenderlos es utilizarlos.
