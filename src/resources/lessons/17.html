-AREN

En esta lección vamos a ver la declinación para hacer el posesivo de una palabra:

Para nombres propios se forman siguiendo esta tabla:

Consonante -EN

Manuel => ManuelEN

Mikel => MikelEN

Aitor => AitorrEN

Vocal -REN

Amaia => AmaiaREN

Alba => AlbaREN

Salomé => SalomeREN

Veamos algunos ejemplos de esto:

GAZTELANIA EUSKARA
Ese es el perro de Amaia Hori da Amaiaren txakurra
¿Cuál es el libro de Mikel? Zein da Mikelen liburua?
El vestido de Esther es muy bonito Estherren soinekoa oso polita da

En el caso de los nombres comunes la tabla es la siguiente:

Singular -AREN

Txakur => TxakurrAREN

Katu => KatuAREN

Zuhaitz (árbol) => ZuhaitzAREN

Plural -EN

Txakur => TxakurREN

Katu => KAtuEN

Zuhaitz => ZuhaitzEN

Veamos algunos ejemplos:

GAZTELANIA EUSKARA
La fruta del árbol Zuhatzaren fruta
Non dago txakurraren janaria? ¿Dónde está la comida del perro?
El juego de los niños es divertido Haurren jolasa dibertigarria da
